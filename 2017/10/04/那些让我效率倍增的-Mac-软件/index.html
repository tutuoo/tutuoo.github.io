图图流着口水盯上了 Mac 软件
以下分享的很多都需要收费，公众号回复『软件』，提供破解版本，若没有对应软件可能是免费，可能遗漏，有实力的童鞋请支持正版！
回复『链接』 有惊喜。
若有出错，请后台指正，目前暂未开通留言，谢谢！

1. Paste

食用效果：一个漂亮的记录复制粘贴工具，可以保留图文，自定义标签组，好用又漂亮~

食用方式：Command + 数字1~9 就是粘贴对应的标签。

2. MindNode

食用效果：一个极简的思维导图工具，简约，小巧，各种类型的图，也是美美哒。

食用方式：打开即食。

3. Alfred 3

黑帽子，我相信知道的人很多，用的人也很多，功能太强大了，有了他，自带的Spotlight 早就被我丢了…

食用效果：
小到可以直接在搜索框算一些简单的算术

大到配合 Dash 直接搜索 API 文档。( 什么是 Dash? 下一个就介绍了)

如果你现在想买一个东西，比方说在淘宝或者京东，一般情况是：
打开浏览器 -> 输入链接，或者打开书签 - > 然后在搜索栏输入你要的。
而现在我们只需要 打开 Alfred 3 输入你想要买的就够了。
虽然仅仅只是把一些过程省略。
但是时间就是金钱。
下面我分别进行了，淘宝搜索，京东搜索，和维基百科的查询。
别问我为什么查零食，我真的不是吃货（捂脸）！

关于如何配置搜索：来张配图。

想自定义其他网站，就去那个网站搜索一个东西，然后把搜索关键字的位置改成 『 {query} 』就好了。

食用方式：此物功能太多，我也只食用了冰山一角，但是已经提高了不少效率，各位吃友加油!

4. Dash

食用效果：一个API文档浏览器和代码段管理器。 Dash帮助您存储代码片段，以及即时搜索和浏览您可能使用的几乎任何API的文档。

如上图所示，我下载了 Android Css 和 JavaScript。
上边的动图就是我通过自定义关键字 ad 为 Android 来查询 Android 相关 API

食用方式：配合上面的 Alfred 3 美滋滋 效果很强大，待挖掘。

5. ScreenFloat

食用效果：一个方便的截屏工具，他可以让你截取的内容置顶，对于多桌面来说是个很好用的工具，可以在一个桌面截图，另一个桌面参照。有自己独立的仓库可以增删。

食用方式：开盖即食。

6. CheatSheet

食用效果：可以查看任意软件的热键，极其方便。

食用方式：长按 Command 即可。

7. Karabiner-Elements

食用效果：爱折腾键盘键位可以试试，我自己把中英文切换键换成了 delete 这样擦除就不需要把手移过去，直接小拇指处理掉，贼方便。还有类似自定义组合件配合『J』『L』变成左右，『I』『,』变成上下，这样右下角的小键盘也可以忽略掉了~

食用方式：单键可以直接改，组合键要自己去定义文件。

8. Tickeys (支持 Windows)

食用效果：可以让键盘模拟多种声音，享受打击快感。

食用方式：开盖即食。

9. Flux (支持 Windows)

食用效果：保护眼睛必备。会跟随作息来调整屏幕色度。

食用方式：开盖即食。

10. Bartender 2

食用效果：可以把状态栏的小图标管理起来，隐藏或者收缩，这样就可以使得界面更加整洁了。

我们正常情况下可能是上面这种情况。
如果我们使用了 Bartender 图片就变成了这样

一个小眼镜 ico 就涵盖了所有的图标，点击可以展开来。当然 ico 的图标是可以更换的。

食用方式：开盖即食。

11. 奇妙清单 (支持 Windows)

食用效果：一个记录事件，可设置时间的日常软件，可以把事情记录上去，到设定的时间点会提醒，支持手机 app 非常方便。

食用方式：事件可以分类型，完成时间。完成之后可以把他勾选掉，给人一种玩游戏把任务刷完的感觉，哈哈哈，满满成就感！

12. Moom

食用效果：一款用来改变当前窗口大小，位置的软件。左对齐，右对齐，居中，一键生效!

食用方式：开盖即食。

13. Manico

食用效果：一款自定义软件打开快捷键的软件，有了他再也不用把鼠标移到应用列表去了，一键打开，任意 DIY。也是我唯一正版的软件（捂脸）

食用方式：开盖即食。

14. LICEcap (支持 Windows)

食用效果：一款录制任意大小 GIF 图的软件，小巧又方便。上面的动图我就是用这个录制的。

食用方式：开盖即食。

15. TG-pro

食用效果：可以实时监控电脑各项性能指标，也可以手动开启风扇。

食用方式：开盖即食。

16. MWeb

食用效果：写作 Markdown 用，没用过别的，感觉挺不错的，还可以一键上传到印象笔记中，反正我觉得挺好用的。

食用方式：开盖即食。

17. 常用的点个名

CleanMyMac，SourceTree，Sublime Text 不多说啦~

最后祝大家中秋节快乐~
