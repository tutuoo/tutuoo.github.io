1. 印象笔记-剪藏

食用效果：可以一键截图网页正文，或者整个网页，部分区域，任你选，然后保存到你指定的文件目录。 效果不要太赞~网页版看到一半，一键保存，出门拿起手机继续看，当然还有更多功能~
食用方式：配合印象笔记 PC 版本或者手机版
（ps：码字的时候一个同事推荐了印象笔记手机版，说其中有个功能可以把手写在纸上的图文更清晰的转化成图片保存起来，用来复习啊，打印啊都是个不错的选择）

2. Vysor

食用效果：一款 （通过USB，不支持Wi-Fi连接）共享手机屏幕在电脑上的插件，可以通过电脑点击，触摸，来控制手机，实现无延迟操作。

如上图所示，可以直接通过电脑桌面来控制手机（注意小米手机可能需要打开触摸相关权限），破解版本和不破解的区别就在于清晰度了。上图我们通过设置修改了画质。

食用方式：因为完整版本需要收费，下面提供破解版本，下载完打开 Chrome 管理插件页面，把压缩包解压导入即可。
食用地址：点击下载

3. 网盘助手


食用效果：可以方便的把网盘的下载地址导出到aria2/aria2-rpc，支持YAAW，然后通过第三方下载工具下载，如迅雷，uTorrent。
食用方式：通过 Google 商店来下载(需要科学上网)

4. AdBlock


食用效果：最受欢迎的Chrome扩展，拥有超过4000万用户！屏蔽整个互联网上的广告。
食用方式：通过 Google 商店来下载(需要科学上网)

5. Isometric Contributions


食用效果：

以下截图取自 Android 之神 『Jake Wharton』
这是我们正常看到的 Github 上 提交的信息情况

下图是用了插件后的效果

是不是很壮观，没错，这插件就是看看用的。

食用方式：通过 Google 商店来下载(需要科学上网)

6. JSON-handle


食用效果：

很漂亮的格式化 Json 数据格式插件~~
食用方式：通过 Google 商店来下载(需要科学上网)

7. LastPass


食用效果：还在为登录各大网站的账号密码而发愁么? 不妨来试试 LastPass，他可以帮助我们记住所有登录过的网站，下次就可以直接登录啦~~
食用方式：通过 Google 商店来下载(需要科学上网)

8. Octotree


食用效果：
有时候我们查看 Github 项目时只想看看一部分内容，又不想下下来，一层层点进去又太繁琐，此插件可以使得浏览器的左边出现侧边栏，来进行各个文件的查看。

食用方式：通过 Google 商店来下载(需要科学上网)

9. OneTab


食用效果：
有时候标签页开了很多，又舍不得关，下次又想用，这时候 OneTab 就起到作用了。
他可以保存便签页，想恢复就恢复，下次打开又满状态复活！

食用方式：通过 Google 商店来下载(需要科学上网)

10. Postman


食用效果：
接口调试工具，反正我觉得挺好用的，一直在用~~

食用方式：通过 Google 商店来下载(需要科学上网)

11. Vimium


食用效果：
此乃神器是也，有了他浏览网页再也不需要鼠标了~~

安装完之后，通过 『shift + ? 』 唤出帮主页面
平常上网的时候
『J』 往下
『K』往上
『D』往下一段
『U』往上一段
『GG』跳到顶部
『Shift + G』跳到底部
『F』之后按提示跳转网页
配合 Shift 按会使网页进入后台打开
当然还有更多的小技巧~~

食用方式：通过 Google 商店来下载(需要科学上网)

12. 掘金


食用效果：
可以让你的 tab 标签页不在单调~

目前有以下版块可以选择。程序猿必备啊

食用方式：通过 Google 商店来下载(需要科学上网)

13. Proxy SwitchyOmega


食用效果：轻松快捷地管理和切换多个代理设置。 科学上网必备。

食用方式：通过 Google 商店来下载(需要科学上网)

14. 斗鱼HTML5播放器


食用效果：斗鱼不支持 HTML5播放，有了此插件后，电脑再也不烫了~ 哈哈给斗鱼打波广告
